# Code of Conduct

This project follows a direct, work-focused code of conduct.

## Standards

- Be respectful and concrete.
- Keep discussion tied to improving First Frame Agent.
- Prefer reproducible reports, minimal examples, and specific proposals.
- Do not post harassment, hate, threats, spam, or private information.

## Enforcement

Maintainers may edit, hide, or remove comments and issues that violate these
standards. Repeated or severe violations may lead to a ban from the project.
For private reports, contact Omid Saffari through the GitHub profile linked in
the README.
